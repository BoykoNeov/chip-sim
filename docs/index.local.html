chip-sim — the gallery

Process recipe in, device out. An educational microchip-fabrication simulator.

Every fab step ships a self-contained demo that prints a cited validation table and banks a figure. This page is the clickable front door to all of it: the visualizations (click any figure for full size), the demos that produce them (run command + source), the v1.x experiments, and the guided notebook. The figures below are pre-committed, so the gallery browses without running anything.

# clone, install the figure stack, run any demo
pip install -e ".[viz]"
python -m chip.demo_junction   # prints the table, banks docs/figures/chip-junction.png
