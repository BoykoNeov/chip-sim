J1

The staged sand→chip journey (phases 1–5): purify a feed (refine → the graded Na edge ring), grow the boule (the two-sided Voronkov pull window — leakage rim ↔ void core), cut a wafer (where down the boule — a graded Scheil V_t centre core, how deep set by the phase-2 pull), diffuse the S/D (the predep dose → R_s → a series-resistance-starved, centre-weighted I_Dsat core), then grow the gate oxide (the two-sided t_ox window — too thin → an edge ring, too thick → a centre core, the only stage failing at opposite radii) — commit each decision and finish to a scored wafer.
