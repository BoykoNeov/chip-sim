chip-sim — the roadmap

Planned, not yet built. The future slices, triaged by the named consumer that would earn each one its build.

Everything on the other three pages is built — every figure there is the banked artifact of a cited, tested demo. This page is the opposite and says so loudly: each card below is a slice from the roadmap triage (future-steps.md) that does not exist in the code yet. The images are schematic previews — hand-drawn illustrations of what the slice would teach, stamped in-image and ribboned here, never simulator output. Each card carries the plan’s verdict verbatim: what observable the slice would add, and the consumer its build is gated on — the repo’s no regime without a named consumer bar, made browsable. When a slice ships, its card comes off this page and its real figures join the galleries.

Promotable — the consumer is named

Each of these already has a device observable waiting to read it; the plan's recommended sequence is F3/F4 (the new-output decision) then F5.

planned — schematic
F3 · 2007 · 45 nm Promotable

High-κ / metal gate. The oxide stage's modern successor: SiO₂ stopped scaling at ~1.2 nm because direct tunneling explodes; HfO₂ holds the same EOT at ~3× the physical thickness.

Would add

A gate-tunneling-leakage observable vs t_ox — a NEW output, plus the SiO₂→high-κ era contrast on the existing oxide/t_ox machinery.

Gated on

Nothing structural — the first genuinely new-output decision (F3 vs F4 order is the open call in the plan).

planned — schematic
F4 · 1997 · Cu damascene Promotable

BEOL interconnect (RC delay). The back end the sim has never had: subtractive Al → Cu dual-damascene → Ru, with chip speed set by wire RC rather than the transistor.

Would add

Delay ∝ R_wire·C_wire — the first output the transistor chain does not set; the richest remaining history arc. Also gives CMP (F8) its missing consumer.

Gated on

Nothing structural — the biggest build of the promotable three.

planned — schematic
F5 · ~2004 · 90 nm Promotable

SiGe strained source/drain. The strain era: embedded SiGe pockets squeeze the p-channel (~2 GPa at ~20% Ge), and hole mobility — hence drive current — rises.

Would add

Mobility → I_Dsat through a strain-aware µ model — the channel-strain rung of the era ladder.

Gated on

A µ(strain) mobility model in device.py — none exists yet; advanced-node, so it queues behind F3/F4.

Coupled & partly built

Real physics whose consumer is (partly) already served — built where the consumer exists, deferred where it would duplicate one.

planned — schematic
F6 · bipolar epi · CMOS wells Coupled to F1

Epitaxy (buried layer / retrograde well). An epitaxial layer over a doped substrate makes buried layers and retrograde wells — profiles a surface predep cannot reach.

Would add

The retrograde/buried-peak profile — but ion implantation (F1, BUILT) already delivers exactly that contrast.

Gated on

Overlaps F1's consumer: a standalone epi slice stays deferred until it would discriminate something implant does not.

F7 · 1998 · STI Partly built (B5)

Isolation remainder: STI process + latchup. LOCOS's bird's beak and the min-active-pitch wall are BUILT (history-mode B5, the 2-D engine's 2nd consumer); the successor's own process is not.

Would add

The STI trench etch/fill step itself and a latchup electrical observable — the two named leftovers of the isolation arc.

Gated on

A consumer for trench geometry / a latchup observable; the beak physics already shipped, so only the remainder is on the roadmap.

Deferred — no discriminating consumer yet

The honest NO’s, kept visible: each is fenced behind a named consumer that does not exist today, with its promotion trigger recorded.

planned — schematic
F8 · enables Cu damascene Deferred — post-F4

CMP / dishing / planarity. Preston-class removal with pattern-density dishing and erosion — post-CMP thickness non-uniformity.

Would add

Post-CMP layer-thickness non-uniformity — IF something read a layer thickness.

Gated on

No consumer today: nothing in the device or yield path reads a layer thickness (backlog D2). Re-evaluate after F4 makes wire cross-section an RC observable.

planned — schematic
F9 · 2011 FinFET · 2022 GAA Deferred — needs 3-D (B1)

FinFET / gate-all-around. The gate wraps the channel — 3 sides (fin) then all 4 (nanosheets) — so the dopant/potential field is non-separable in three dimensions.

Would add

Width-dependent V_t / corner effects a 3-D depletion field carries — the consumer that would finally promote the engine's last deferred regime.

Gated on

The 3-D engine (backlog B1, trigger recorded): 1-D depth + the 2-D cross-section serve every current device; 3-D waits for this consumer, exactly as 2-D waited for v1.8.

planned — schematic
F10 · 2019 · 7 nm Deferred

EUV / multipatterning. The wavelength ladder's last rung (λ = 13.5 nm) and pitch-splitting (LELE) — the current frontier of the litho arc.

Would add

Nothing the model does not already have: litho.py computes the aerial image, R = k₁λ/NA, defocus, PEB and CAR down the whole ladder.

Gated on

No discriminating consumer — a shorter λ or a split pitch adds no new observable (the plan's honest NO).
